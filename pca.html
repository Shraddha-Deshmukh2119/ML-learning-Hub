Principal Component Analysis (PCA)

Dimensionality reduction technique for finding patterns in data

What is Principal Component Analysis?

Principal Component Analysis (PCA) is a statistical procedure that uses an orthogonal transformation to convert a set of observations of possibly correlated variables into a set of values of linearly uncorrelated variables called principal components.

In simpler terms, PCA:

  • Identifies patterns in data based on the correlation between features
  • Reduces the dimensionality of the data while preserving as much variability as possible
  • Transforms data to a new coordinate system where the new axes (principal components) represent directions of maximum variance

The first principal component accounts for as much of the variability in the data as possible, and each succeeding component accounts for as much of the remaining variability as possible while being orthogonal to the preceding components.

Steps in PCA:

  1. Standardize the data (mean=0, variance=1)
  2. Calculate the covariance matrix
  3. Calculate eigenvectors and eigenvalues of the covariance matrix
  4. Sort eigenvectors by decreasing eigenvalues
  5. Choose k eigenvectors with the largest eigenvalues
  6. Transform the original data to obtain k principal components

Applications:

  • Dimensionality reduction for visualization
  • Noise filtering
  • Feature extraction
  • Data compression
  • Preprocessing for machine learning algorithms
