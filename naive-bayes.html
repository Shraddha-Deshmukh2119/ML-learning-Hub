Naive Bayes

Probabilistic classifiers based on Bayes' theorem

What is Naive Bayes?

Naive Bayes is a family of simple probabilistic classifiers based on applying Bayes' theorem with strong (naive) independence assumptions between the features. Despite their simplicity, Naive Bayes classifiers often work surprisingly well in many real-world situations, especially in document classification and spam filtering.

The "naive" part comes from the assumption that features are conditionally independent given the class label. In reality, this assumption rarely holds true, but the algorithm often performs well regardless.

Bayes' theorem provides a way to calculate the probability of a hypothesis based on prior knowledge. The formula is:

P(A|B) = P(B|A) × P(A) / P(B)

For classification, we apply this to find the most likely class (c) given the features (x):

P(c|x) = P(x|c) × P(c) / P(x)

Types of Naive Bayes Classifiers:

  • Gaussian Naive Bayes: Assumes that features follow a normal distribution
  • Multinomial Naive Bayes: Suitable for features representing discrete counts (e.g., word counts for text classification)
  • Bernoulli Naive Bayes: Used when features are binary (0/1) values

Advantages:

  • Simple and easy to implement
  • Works well with high-dimensional data
  • Requires relatively small amount of training data
  • Fast training and prediction
  • Not sensitive to irrelevant features

Applications:

  • Spam filtering
  • Document classification
  • Sentiment analysis
  • Recommendation systems
  • Medical diagnosis
