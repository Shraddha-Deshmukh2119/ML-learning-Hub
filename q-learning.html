Q-Learning

A model-free reinforcement learning algorithm

What is Q-Learning?

Q-Learning is a model-free reinforcement learning algorithm that learns the value of an action in a particular state. It's "model-free" because it doesn't require a model of the environment and can handle problems with stochastic transitions and rewards without requiring adaptations.

Key components of Q-Learning:

  • Q-value: The expected future reward for taking action a in state s
  • Q-table: A lookup table where each state-action pair maps to a Q-value
  • Learning rate (α): How much new information overrides old information
  • Discount factor (γ): The importance of future rewards
  • Exploration vs. Exploitation: Balance between trying new actions and using known good actions

The Q-Learning update rule is:

Q(s, a) ← Q(s, a) + α[r + γ·maxa'Q(s', a') - Q(s, a)]

Where:

  • s is the current state
  • a is the action taken
  • r is the reward received
  • s' is the new state
  • maxa'Q(s', a') is the maximum Q-value for the new state across all possible actions

Applications:

  • Game playing (e.g., Atari games)
  • Robot navigation
  • Resource management
  • Traffic light control
  • Portfolio optimization
