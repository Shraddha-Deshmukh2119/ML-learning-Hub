Supervised Learning

Learning from labeled data to make predictions

What is Supervised Learning?

Supervised learning is a machine learning paradigm where the algorithm learns from labeled training data. The algorithm makes predictions based on that data and is corrected when predictions are wrong. The learning stops when the algorithm achieves an acceptable level of performance.

In supervised learning, each example is a pair consisting of:

  • An input object (typically a vector of features)
  • A desired output value (also called the supervisory signal)

The supervised learning algorithm analyzes the training data and produces an inferred function, which can be used for mapping new examples. An optimal scenario allows the algorithm to correctly determine the class labels for unseen instances.

Common Supervised Learning Algorithms:

  • Linear Regression
  • Decision Trees
  • Neural Networks
  • Support Vector Machine
  • Naive Bayes
  • K-Nearest Neighbour
